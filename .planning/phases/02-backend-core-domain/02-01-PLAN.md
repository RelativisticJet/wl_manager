**Public API:**
1. `read_csv(filepath: str) -> Tuple[List[str], List[Dict]]` — Read CSV file, return (headers, rows). Raise OSError on missing file.

2. `write_csv(filepath: str, headers: List[str], rows: List[Dict]) -> None` — Write CSV file atomically (write to temp, rename). Raise OSError on permission denied.

3. `compute_diff(old_headers: List[str], old_rows: List[Dict], new_headers: List[str], new_rows: List[Dict]) -> Dict` — Compare old vs new CSV and return structured diff with keys:
   - `added`: list of new row dicts
   - `removed`: list of deleted row dicts
   - `edited`: list of dicts with {old_row, new_row, row_num, changed_fields}
   - `added_count`, `removed_count`, `edited_count`: int counts
   - `added_columns`, `removed_columns`: list of column names
   - `text_diff`: list of unified-diff style lines

   Algorithm (lines 1721-1870 from handler):
   - Use Counter (multiset) for duplicate row detection, not sets
   - Reverse iteration on added_raw to pick actually-new rows (frontend appends at end)
   - Similarity-based matching for edit detection (require >50% fields unchanged)
   - Skip edit detection if either side exceeds MAX_DIFF_ROWS (O(n²) guard)
   - Use object id() for distinguishing same-content rows, not key-based matching

4. `get_expire_column(headers: List[str]) -> Optional[str]` — Find "Expires" column (case-insensitive). Return column name or None.

5. `remove_expired_rows(headers: List[str], rows: List[Dict], tz_offset_minutes: int = 0) -> Tuple[List[Dict], int]` — Remove rows where Expires column is in the past (adjusted by tz_offset_minutes). Return (filtered_rows, removed_count).

6. `get_column_widths(csv_path: str) -> Dict[str, int]` — Read column widths JSON file (side-car to CSV). Return dict {col_name: width_px} or empty dict if file missing.

7. `set_column_widths(csv_path: str, widths: Dict[str, int]) -> None` — Write column widths JSON file. Create if missing, overwrite if exists.

**Error Handling:**
- read_csv: raise OSError if file missing or unreadable
- write_csv: raise OSError if permission denied; use atomic write (temp → rename)
- compute_diff, get_expire_column, remove_expired_rows: no exceptions, return safe defaults (empty lists, None, 0)
- get/set_column_widths: silently return empty dict or skip on file errors (non-critical feature)

**Type Hints:**
All functions have full type hints. Use `Optional[T]` for return values that can be None.
