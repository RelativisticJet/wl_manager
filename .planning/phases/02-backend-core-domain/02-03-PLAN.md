**Public API:**

1. `get_versions_dir(csv_path: str) -> str` — Return absolute path to _versions/ directory for a CSV. Create if missing. Return path string.

2. `read_version_manifest(csv_path: str) -> Tuple[Dict, str]` — Read version manifest JSON file ({csv_name}_versions.json). Return (manifest_dict, error_string).

   Manifest dict structure:
   ```json
   {
     "csv_file": "DR102_whitelist.csv",
     "current_version": "20260331_203045",
     "versions": {
       "20260331_203045": {
         "timestamp": "2026-03-31 20:30:45",
         "analyst": "admin",
         "action": "save",
         "row_count": 42,
         "hash": "abc123..."
       },
       ...
     }
   }
   ```

   Return (manifest, "") on success, ({}, error_msg) on failure (file missing, invalid JSON).

3. `write_version_manifest(csv_path: str, manifest: Dict) -> Tuple[bool, str]` — Write version manifest JSON file atomically. Return (success, error_string).

4. `snapshot_version(csv_path: str, analyst: str, action_label: str = "save") -> Tuple[str, str]` — Create a new version snapshot (timestamped copy of CSV). Return (timestamp_id, error_string).

   Steps:
   1. Read current CSV (call read_csv from wl_csv)
   2. Get versions directory
   3. Generate timestamp ID (format: "%Y%m%d_%H%M%S")
   4. Create snapshot file: {versions_dir}/{csv_name}_{timestamp_id}.csv
   5. Write headers and rows to snapshot
   6. Update manifest: add version entry with timestamp, analyst, action, row_count
   7. Enforce MAX_VERSIONS limit: keep only newest MAX_VERSIONS versions (delete oldest snapshots and manifest entries)
   8. Write updated manifest

   Return (timestamp_id, "") on success, ("", error_msg) on failure.

5. `get_versions_list(csv_path: str) -> Tuple[List[Dict], str]` — Get list of all versions for a CSV (for revert dropdown). Return (versions_list, error_string).

   Each item in list:
   ```python
   {
     "version_id": "20260331_203045",
     "timestamp": "2026-03-31 20:30:45",
     "analyst": "admin",
     "action": "save",
     "row_count": 42,
     "display": "31-03-2026 20:30:45 (42 rows, by admin)"
   }
   ```

   Sorted newest-first (reverse chronological). Return (list, "") on success, ([], error_msg) on error.

**Error Handling:**
- read_version_manifest: return ({}, error_msg) on file missing or JSON invalid
- write_version_manifest: return (False, error_msg) on permission denied or write failure
- snapshot_version: return ("", error_msg) on any failure (read CSV, write snapshot, update manifest)
- get_versions_list: return ([], error_msg) on manifest missing/invalid
