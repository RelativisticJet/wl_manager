Create bin/wl_audit.py (~150 lines) exporting 3 public functions for audit event management.

**Imports:**
```python
import json
from datetime import datetime

from wl_constants import AUDIT_INDEX, AUDIT_SOURCETYPE, AUDIT_SOURCE
from wl_logging import get_audit_logger
```

**Public API:**

1. `get_audit_logger() -> logging.Logger` — Return configured audit logger. Call through to wl_logging.get_audit_logger().

2. `build_audit_event(action: str, analyst: str, detection_rule: str, csv_file: str, **kwargs) -> Dict` — Construct structured audit event dict with all required fields. Return event dict ready for posting.

   Required parameters:
   - action: "added", "removed", "removed_multiple", "edited", "revert", "auto_removed"
   - analyst: username performing action
   - detection_rule: detection rule name
   - csv_file: CSV filename
   - **kwargs: Additional fields (comment, removed_row_count, edited_row_count, etc.)

   Returns dict with keys:
   ```python
   {
     "timestamp": "2026-03-31 20:30:45",
     "action": "save",
     "analyst": analyst,
     "detection_rule": detection_rule,
     "csv_file": csv_file,
     "app_context": kwargs.get("app_context", ""),
     "comment": kwargs.get("comment", ""),
     ... (all kwargs merged in)
   }
   ```

3. `post_audit_event(session_key: str, event: Dict) -> Tuple[bool, str]` — Post audit event to wl_audit index via Splunk REST API. Return (success, error_string).

   Steps:
   1. Call splunk.rest.simpleRequest() to POST /services/receivers/simple
   2. Set host, source, sourcetype, time headers
   3. Send event dict as JSON in request body
   4. Return (True, "") on success (status_code 200-299)
   5. Return (False, error_msg) on failure (4xx/5xx status, network error)

   On error, log to audit logger (via get_audit_logger()) and return error.

**Error Handling:**
- build_audit_event: no exceptions, always returns dict (fills missing fields with defaults)
- post_audit_event: return (success_bool, error_msg) instead of raising; log failures to audit logger

**Type Hints:**
All functions fully type-hinted.

**__all__ Export:**
```python
__all__ = [
    'build_audit_event',
    'post_audit_event',
    'get_audit_logger',
]
```

**Notes:**
- Audit logger is defined in wl_logging.py (Phase 1), so this just wraps/re-exports it
- Event construction is pure function (no side effects)
- Posting is async-friendly (non-blocking HTTP call, failures logged but don't crash handler)
