**Pipeline call patterns:**

```python
def _action_save_csv(self, request, payload, user, roles):
    try:
        csv_file = payload.get("csv_file")
        rows = payload.get("rows")
        expected_mtime = payload.get("expected_mtime")
        comment = payload.get("comment", "")
        
        # Precondition validation
        if not csv_file or not isinstance(rows, list):
            return self._resp(400, {"error": "csv_file and rows required"})
        
        # Approval gate check
        gate_required, gate_reason = wl_approval.check_approval_gate(
            user, "save_csv", {csv_file: csv_file, edit_count: len(rows)}
        )
        if gate_required:
            success, msg = wl_approval.submit_approval(
                user, "save_csv", {csv_file, rows, expected_mtime, comment}, 
                reason=comment, session_key=self.session_key
            )
            if success:
                return self._resp(202, {success: True, message: "Approval requested"})
            else:
                return self._resp(400, {error: msg})
        
        # Execute directly (no approval needed)
        success, message = wl_csv.save_csv_pipeline(csv_file, rows, expected_mtime)
        if not success:
            # Mtime conflict is 409 Conflict
            if "mtime" in message.lower():
                return self._resp(409, {error: message})
            return self._resp(400, {error: message})
        
        # Post audit event
        try:
            event = wl_audit.build_audit_event(
                action="save_csv",
                analyst=user,
                detection_rule=payload.get("detection_rule", ""),
                csv_file=csv_file,
                comment=comment
            )
            wl_audit.post_audit_event(self.session_key, event)
        except Exception as e:
            _logger.exception("Failed to post audit event")
        
        return self._resp(200, {csv_file: csv_file, success: True})
    
    except ValueError as e:
        return self._resp(400, {error: str(e)})
    except FileNotFoundError:
        return self._resp(404, {error: "CSV file not found"})
    except IOError as e:
        return self._resp(500, {error: "File I/O error"})
    except Exception as e:
        _logger.exception(f"Error in _action_save_csv: {e}")
        return self._resp(500, {error: "An internal error occurred"})
```

**Approval gate check pattern:**
- Call wl_approval.check_approval_gate(user, action_type, payload) → (gate_required, reason)
- If gate_required: submit_approval and return 202 (Accepted)
- If no gate: execute directly
- Never proceed with direct execution if gate is required and approval not checked

**Error codes:**
- 200: Success
- 202: Accepted for approval (return immediately, don't execute)
- 400: Validation error or bad request
- 404: CSV/rule not found
- 409: Conflict (e.g., mtime mismatch)
- 403: Permission denied (caught in _dispatch RBAC)
- 500: Server error
