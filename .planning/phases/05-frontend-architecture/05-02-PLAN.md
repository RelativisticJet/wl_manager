## Locked Decisions from CONTEXT.md

### Module Boundaries
- **Feature modules depend only on foundation:** wl_search, wl_presence, wl_csv_io require only wl_constants, wl_state, wl_rest, wl_ui
- **No cross-feature dependencies:** Communication via State events and wl:* custom events
- **wl_csv_io is single module:** Both import (parser, validator, preview) and export in one module (~300 lines)

### Search Module (wl_search.js)
- Text search across all visible columns
- Returns filtered rows matching search term
- Listens to currentRows State changes to update available columns
- Clear button to reset search

### Presence Module (wl_presence.js)
- Poll backend for who is currently editing this CSV
- Heartbeat to announce own presence
- Display presence indicator in UI (e.g., "admin is editing")
- Update State.set('userPresence', {username: role, ...})

### CSV I/O Module (wl_csv_io.js)
- Export current rows as downloadable CSV file
- Import CSV file via file picker
- Parse CSV, validate structure (headers match current CSV)
- Update State with imported rows
