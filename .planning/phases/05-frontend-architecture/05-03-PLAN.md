## Locked Decisions from CONTEXT.md

### wl_table.js (large module, acceptable)
- Table rendering, cell editing, pagination, column resize, drag-drop, date picker, undo bar, row selection
- ~1500-2000 lines is acceptable — all tightly coupled to table DOM
- Module-local state: currentPage, ROWS_PER_PAGE, resizeState, dragState, colWidths, undoState
- Pagination: currentPage and page navigation are module-local (not in State)
- Column resize + drag-drop: resizeState, dragState, colWidths are table-specific
- Date picker: activated by clicking Expires column cell
- refreshTable() auto-calls syncInputs() first (structural enforcement)

### wl_modals.js
- Add/remove/edit dialogs with form validation
- Remove requires reason (per-row or bulk)
- Per-row Comment column support

### wl_versions.js
- Dropdown showing last 5 versions with "Current" label
- Revert functionality with version selection
- Format: "24-02-2026 12:37:16 (42 rows, by admin)"

### wl_approval_ui.js
- Display approval queue for admin
- Show approval status for user's requests
- Daily limit message formatting

- Listen to 'state:currentRows' to refresh on external changes
- syncInputs() captures any unsaved DOM input values into currentRows before refreshTable()
- refreshTable() calls syncInputs() first (structural enforcement)
- Row numbers from position in currentRows (not persisted)
