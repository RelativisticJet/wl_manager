- `history.replaceState(state, title, url)` for URL updates without page reload

- Access denied (403): Show alert "You do not have permission to access the Control Panel. Administrators only." and return early (no modules loaded)
- Module init throws: Catch exception, call `UI.showFatalError()`, stop (fail-fast pattern)
- Network failures: Pass error to showCpAlert with HTTP status code included
