## Dependency Injection via ctx Object

Entry point passes context:
```javascript
var ctx = {
  showAlert: showCpAlert,       // Show alerts
  showConfirm: showCpConfirm,   // Confirmation modals
  showPrompt: showCpPrompt,     // Input prompts
  currentUser: cpCurrentUser,   // Username
  isSuperAdmin: cpIsSuperAdmin, // Superadmin flag
  isAdmin: cpIsAdmin            // Admin flag
};
```

All modules receive ctx and use:
- ctx.showAlert() for errors
- ctx.showConfirm() for destructive actions (restore, purge)
- ctx.currentUser for audit logging
- ctx.isSuperAdmin for access control (Admin Limits)

- wl_cp_trash.js: trashItems, currentPage, pollingInterval, loadedRetention
