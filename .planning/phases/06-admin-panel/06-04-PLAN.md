## Test Coverage Scope

Each module test file should verify:
1. **Module load:** `require()` succeeds, module defines expected API
2. **init() call:** Module initializes with context object, caches DOM elements
3. **load() call:** Module fetches data, renders table/form
4. **Event handlers:** Click handlers, input change handlers, pagination work
5. **Polling (if applicable):** startPolling/stopPolling functions exist and manage intervals
6. **Error handling:** Network failures show alerts via ctx.showAlert
7. **Modal guard (if applicable):** Polling skips when modal open
8. **Data presentation:** Rendered HTML matches expected structure (columns, buttons, pagination)
